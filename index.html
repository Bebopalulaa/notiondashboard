Configure ton dashboard

  1. Créer une intégration Notion
    Va sur notion.so/my-integrations et crée une intégration interne.
  2. Copier le token
    Copie l'"Internal Integration Token" (commence par secret_).
  3. Partager ta base Notion
    Ouvre ta base → ··· → Connections → ajoute ton intégration.
  4. Copier l'ID de la base
    Dans l'URL : notion.so/{database_id}?v=...
  5. (Optionnel) Déployer le proxy CORS
    Consulte worker/README.md — recommandé.
